DOM Manipülasyonu

DOM (Document Object Model), HTML dökümanının JavaScript ile etkileşimli olarak kullanılmasını sağlayan yapıdır. Bu bölümde DOM üzerinde nasıl seçim yapılır, içerik nasıl değiştirilir ve yeni öğeler nasıl eklenir gibi temel konulara değineceğiz.

Element Seçimi

const baslik = document.getElementById("title");
const paragraflar = document.getElementsByClassName("text");
const secilen = document.querySelector(".menu li:first-child");

İçeriği Değiştirme

baslik.innerText = "Yeni Başlık";
paragraf.innerHTML = "Kalın yazı";

Yeni Eleman Ekleme

const yeniLi = document.createElement("li");
yeniLi.textContent = "Yeni Menü";
document.querySelector(".menu").appendChild(yeniLi);

Uygulama Önerisi

Bir listeye kullanıcıdan alınan veriyi ekleyen basit bir uygulama yap. `createElement`, `appendChild`, `value` gibi özellikleri kullanmayı unutma.
